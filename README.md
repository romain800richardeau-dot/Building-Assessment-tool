# Verdict — Building Performance Codex

> Outil d'évaluation de la performance environnementale du bâtiment.
> Multi-référentiel : BREEAM NC V7, BREEAM RFO 2015, HQE, BBCA, PLU.

🔗 **Site live : [bat-verdict.fr](https://bat-verdict.fr)**

---

## Vue d'ensemble

Verdict est une plateforme conçue pour simplifier le quotidien des professionnels
de la construction durable — AMO, MOE, MOA, bureaux d'études — en rendant le suivi
documentaire des certifications environnementales plus dynamique et interactif.

**Sept modules :**

1. **Multi-référentiel** — Une saisie partagée, cinq verdicts (BREEAM NC V7, BREEAM RFO 2015, HQE, BBCA, PLU)
2. **Atelier cartographique** — Couches géographiques (risques, géologie, climat, écosystèmes…), annotations live et export PDF
3. **Site analysis live** — 9 thèmes branchés en direct sur BRGM, IGN, Natura 2000, ADEME, données météo
4. **Bonnes pratiques** — 15 thèmes, ~135 fiches au format manuel scolaire
5. **Actions MOA** — Chaque crédit traduit en geste concret pour le Maître d'Ouvrage
6. **Catalogue biosourcé** — 5 000+ FDES INIES, Base Carbone ADEME, 727 acteurs réemploi, 5 343 PMCB
7. **Méthodologie & preuves** — Compliance notes bilingues FR + EN, méthodologie pas-à-pas, templates de preuves

---

## Stack technique

- HTML / CSS / JavaScript vanille — application monopage
- Hébergé sur GitHub Pages
- Domaine custom : `bat-verdict.fr`
- Aucun framework lourd, aucun backend côté client (privacy-first, données stockées en local)
